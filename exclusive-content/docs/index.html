The <wm-exclusive-content> component

The <wm-exclusive-content> is a web component that you can use to show exclusive content if it determines the user is sending payments via web monetization.

The component only activates if the Web Monetization API is available, otherwise it will show nothing.

The component has 1 attribute you can use:

template
A CSS selector for a <template> element. The component will find the template, hydrate it and use it as the exclusive contents. This overrides nesting a template element within the custom element.

Examples

If you are using a browser that supports the Web Monetization API then you will see the text "Thank you for supporting this site with Web Monetization" below once you start sending payments. If your browser supports Web Monetization and you choose not to send a payment to the page then the text will not appear. If your browser doesn't support Web Monetization then you will not see the text at all. 

<wm-exclusive-content>
  <template>
    <p><strong>Thank you for supporting this site with Web
    Monetization</strong></p>
  </template>
</wm-exclusive-content>
